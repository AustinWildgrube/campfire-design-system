<h2 className="h2">Sizes</h2>

Right now, Campfire is using a scale that ranges from `xxs` to `xxxl`. Each level has a different pixel value associated with it. The scale is:

- `xxs` - `4px`
- `xs` - `8px`
- `sm` - `12px`
- `md` - `16px`
- `lg` - `20px`
- `xl` - `32px`
- `xxl` - `44px`
- `xxxl` - `64px`

<h2 className="h2">Shared Attributes</h2>

All Campfire components support these attributes:

 - Margin:
    - `usiM` – sets `margin` property on root element.
    - `usiMy` – sets `margin-top` and `margin-bottom` properties on root element.
    - `usiMx` – sets `margin-right` and `margin-left` properties on root element.
    - `usiMt` – sets `margin-top` property on root element.
    - `usiMb` – sets `margin-bottom` property on root element.
    - `usiMl` – sets `margin-left` property on root element.
    - `usiMr` – sets `margin-right` property on root element.

Margin can get value from `theme.spacing` as well:

```html
<Card usiM="xl">Card</card> -> margin: theme.spacing.xl
<Card [usiM]="100">Card</card> -> margin: 100px
```

- Padding:
    - `usiP` – sets `padding` property on root element
    - `usiPy` – sets `padding-top` and `padding-bottom` properties on root element
    - `usiPx` – sets `padding-right` and `padding-left` properties on root element
    - `usiPt` – sets `padding-top` property on root element
    - `usiPb` – sets `padding-bottom` property on root element
    - `usiPl` – sets `padding-left` property on root element
    - `usiPr` – sets `padding-right` property on root element

Padding can get value from `theme.spacing` as well:

```html
<Card usiP="xl">Card</card> -> padding: theme.spacing.xl
<Card [usiP]="100">Card</card> -> padding: 100px
```
