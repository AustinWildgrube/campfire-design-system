# Tabs
Tabs are a set of layered sections of content, known as tab panels, that display one panel of content at a time. Each tab panel has an associated tab element,
that when activated, displays the panel. The list of tab elements is arranged along one edge of the currently displayed panel, most commonly the top edge.
When a tabbed interface is initialized, one tab panel is displayed and its associated tab is styled to indicate that it is active. When the user activates one
of the other tab elements, the previously displayed tab panel is hidden, the tab panel associated with the activated tab becomes visible, and the tab is
considered "active".

```html
<usi-tab-group>
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>

  <usi-tab usiLabel="Tab 2">
    <p>Tab 2 content</p>
  </usi-tab>

  <usi-tab usiLabel="Tab 3">
    <p>Tab 3 content</p>
  </usi-tab>
</usi-tab-group>
```

## Labels
A label is a required attribute of the tab element. It is used to identify the tab element and to display the tab's label. A tab label should **always** be
short and descriptive.

```html
<usi-tab-group>
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
  <usi-tab usiLabel="Tab 2">
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```

## Disabling
Tabs can either be disabled one at a time or can be disabled at the group level. Disabling a single tab is good for when the entirety of the content of that one
panel is not available. When disabling the tab group `usiDisabled` will be put on the `<usi-tab-group>` element. Likewise, when a single tab needs to be disabled,
the `usiDisabled` attribute will be put on the `<usi-tab>` element.

```html
<usi-tab-group usiDisabled>
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>

<usi-tab-group>
  <usi-tab usiLabel="Tab 1" usiDisabled>
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```

## Full Width Tabs
By default, tabs have a `min-width` of `160px` and a `width` of `fit-content`. This means that tabs will not take up the full width of the screen. When the
`usiGrow` attribute is added to the `usi-tab-group` element, the `flex-grow` attribute is added to all child elements.

```html
<usi-tab-group usiGrow>
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```

## Positioning
Tabs can be positioned on 3 different parts of the x-axis. This is the `left`, `center` and `right` positions. The `left` position is the default and will be used
when the `usiPosition` attribute is not set.

```html
<usi-tab-group usiTabPosition="center">
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```

## Active Tabs
Tabs will always load to the DOM with the first one set to active. If a different tab needs to active on load the `usiActiveTab` attribute can be used on that
specific tab.

```html
<usi-tab-group>
  <usi-tab usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
  <usi-tab usiLabel="Tab 2" usiActive>
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```

## Customization
Campfire tabs were made to be easily customizable. Styling a tab group or a single tab is as simple as just adding a class onto the tag or overriding our CSS
variables in the root selector. The list of overridable variables are as follows:

```css
:root {
  --tab-group-border-color: #e6e6e6;

  --tab-group-border-color--active: var(--primary);
  --tab-group-border-height--active: 2px;

  --tab-group-border-color--disabled: var(--gray);

  --tab-label-text-color: #515251;
  --tab-min-width: 160px;

  --tab-label-text-color--active: var(--black);

  --tab-label-text-color--disabled: var(--gray);

  --tab-background-color--hover: #f5f5f5;
}
```

```html
<usi-tab-group class="custom-tab-group">
  <usi-tab class="custom-tab" usiLabel="Tab 1">
    <p>Tab 1 content</p>
  </usi-tab>
</usi-tab-group>
```
